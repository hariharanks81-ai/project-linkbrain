# Link Brain 🧠

A private, fast, and intelligent Progressive Web App (PWA) link manager with offline support, semantic search, and multi-language voice dictation.

## ✨ Features

- **Installable PWA**: Add to your home screen for a native app-like experience with full offline functionality.
- **AI Semantic Search**: Find links using high-level concepts rather than exact keyword matching (runs locally via Xenova Transformers).
- **Voice Dictation**: Dictate your personal notes or search queries out loud. Fully supports English and Tamil.
- **Rich Previews**: Instant visual cards for websites, YouTube videos, and image URLs.
- **Data Privacy**: All data stays on your machine in an encrypted-like IndexedDB format. 
- **Backups**: Export your link base seamlessly as a JSON file or a beautiful PDF summary.

## 🚀 Getting Started

### Running Locally
Since it's a static PWA, you can serve it easily using any standard HTTP server. 

Using Python:
```powershell
python -m http.server 8080
```
Open your browser and navigate to: [http://localhost:8080]


### PWA Installation
To install the application on your mobile device or desktop:
1. Open the page in Chrome/Safari.
2. Tap the floating prompt or access browser options (⋮ / Share).
3. Select **"Add to Home Screen"** or **"Install Link Brain"**.
